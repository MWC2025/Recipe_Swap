# Recipe_Swap
Recipe Swap 

Recipe Swap is a web application for users to share and swap personal recipes in a structured platform. Users register accounts using email and password, log in securely, and manage profiles that display their uploaded recipes and activity history. 

Recipe uploads feature title, description, ingredients list, step-by-step instructions, tags (e.g., vegetarian, quick, desserts), and photo attachments, with options to edit or delete personal content.

The listings page allows for browsing all active recipes through previews, keyword search, and filters by tags or categories, linking to detail views showing creator profiles, average star ratings, user reviews, multiple images, and swap options. 

The swap mechanism supports requests tracking requester ID, offered recipe ID, requested recipe ID, status (pending, accepted, declined), and timestamps for creation/updates.

Ratings use 5 stars with editable reviews, averages guide selections. Moderators remove inappropriate reviews, content, posts, or users.
